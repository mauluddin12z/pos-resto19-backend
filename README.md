# Point of Sale - Warung 19 (Backend)

This is the backend service for the Point of Sale (POS) system developed for Warung 19, a local restaurant. It handles core business logic such as order processing, transaction management, menu data, and database operations.

The backend provides APIs that support the frontend application, ensuring smooth and efficient restaurant operations.

---

## Features

- RESTful API for POS operations
- Menu management (CRUD)
- Order and transaction processing
- Sales data handling
- Database integration (MySQL)
- Structured backend architecture

---

## Tech Stack

- Express.js
- Node.js
- MySQL

---

## API Endpoints

Base URL:

/api

GET /menu
POST /menu
GET /orders
POST /orders
PUT /orders/:id
DELETE /orders/:id


---

## Purpose

This backend is built to support restaurant operations by managing orders, transactions, and menu data in a centralized and efficient system.

---

## Future Improvements

- Authentication & role-based access (Admin / Cashier)
- Real-time order updates (WebSocket)
- Payment gateway integration
- Inventory management system
- API security improvements (JWT, rate limiting)
