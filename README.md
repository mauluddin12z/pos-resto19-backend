# Point of Sale - Warung 19 (Backend)

This is the backend of a Point of Sale (POS) web application designed for Warung 19, a local restaurant. The system handles core business logic such as managing orders, menus, and transactions, ensuring smooth and efficient restaurant operations.

The backend provides APIs that support the frontend application and manages data flow between the application and the database in real time.

---

## Features

- Menu management (create, read, update, delete)
- Order processing system
- Transaction handling
- Sales data management
- RESTful API structure
- Database integration

---

## Tech Stack

- Express.js
- Node.js
- MySQL

---

## Purpose

This backend is built to support restaurant operations by handling data processing, order management, and transaction workflows in a centralized and efficient system.

---

## Future Improvements

- Authentication and role-based access (admin/cashier)
- Real-time order updates using WebSocket
- Payment gateway integration
- Inventory management system
- API security improvements (JWT, rate limiting)
